¶ Before compiling

1) Download a zipfile 2) Decompress it 3) Go inside the folder, and type in

$ sudo npm install

4) Make sure you have 'socket.io', 'sequelize', 'connect-flash', 'model' and 'sqlite3' in your node_modules

5) Run your application

$ sudo node app.js

Then access to http://localhost:3003/. It is important to run node using sudo, because if you didn't, you might get the socket error.

If you get module errors, try

$ npm cache clean
$ npm install

This will change your local setup. Then rerun node.
